# 视觉稿：主界面重做（拍板 ASCII 示意）

> 本文件是 PRD 验收「空态/有图态布局与拍板的 ASCII 示意一致」的参考基准。
> 落码前经用户过目（2026-07-05）。设计语言沿用既有 Apple 风玻璃拟态，不新造风格。
> 图标一律内联 SVG（沿用现有齿轮自绘风格，不引图标库，符合无依赖约定）。

## 共享设计 token（复用既有，不新造）

| 项 | 值 |
|---|---|
| 主渐变 | `linear-gradient(135deg,#0071e3,#6200ff)` |
| 圆角 | 8 / 12 / 14 / 16 / 20 / 99(药丸) |
| 玻璃 | `backdrop-filter: blur(20~40px)` + 半透明白底 |
| 阴影 | 柔和 `0 10px 30px rgba(0,0,0,.05~.1)`；主按钮 `0 5px 15px rgba(0,113,227,.2)` |
| 动效 | 入场弹性 `cubic-bezier(.34,1.56,.64,1)`；态切换 `cubic-bezier(.4,0,.2,1)` 180ms |
| 间距 | 8pt 节奏（8/16/24/32）；值用等宽字体 |

UX 校验点（来自 ui-ux-pro-max）：aria-live 错误、empty-state 引导、active tab 高亮、disabled 半透明、hover 150-300ms、SVG 图标、44px 触达、reduced-motion、断点 768。

## ① 空态 Dropzone

```
┌─────────────────────────────────────────────────────────── [⚙] ─┐
│              高精度 OCR 识别   (渐变文字标题)                      │
│         智能识别多国语言 · 手写 · 表格 · 公式，即拍即取            │
└───────────────────────────────────────────────────────────────┘

      ╔═══════════════════════════════════════════════════╗
      ║                                                   ║
      ║                    ╭─────╮                        ║   虚线边框 2px / 圆角20 / 玻璃底
      ║                    │ 📷  │  ← 相机 SVG(渐变描边64) ║   拖拽态: 边框转实心渐变+内发光+scale(1.01)
      ║                    ╰─────╯                        ║   相机图标 dragging 时轻微上浮
      ║                                                   ║
      ║          拖拽、点击 或 Ctrl+V 粘贴上传             ║   ← 主提示(前景色)
      ║              支持 PNG · JPG · PDF                  ║   ← 副提示(次要色/12px)
      ║                                                   ║
      ╚═══════════════════════════════════════════════════╝

        [ ⬆ 上传 ]  [ 📋 粘贴 ]  [ 🔗 链接 ]  [ ✕ 清除 ]
         主(渐变填充)   幽灵      幽灵      幽灵/禁用(空态)
```

- 点击 dropzone 主体 = 触发文件选择（与「上传」等效）；「链接」点开后在按钮排下方内联展开 url 输入行。
- 按钮：药丸、高 44px、`cursor-pointer`、hover 上浮 1px；空态「清除」`disabled`（opacity .4，无指针）。
- dropzone `role` 可聚焦，focus 显渐变环；空态引导文案即 UX 的 empty-state。

- **紧凑态**（仅 Web ≤768px，桌面 isTauri 不进）：上下堆叠（图上/结果下），分隔条隐藏，缩略图条保留。

## ③ ResultPane：双 tab + 图标按钮 + 重试

```
三种态 ──────────────────────────────────────────────

[原文 ▔▔] [译文 ◌翻译中]     tab：译文仅在「有译文/翻译中」显示；翻译中 spinner，失败显角标 ⚠
──────────────────────────
⎘ 复制    ✎ 格式纠错          图标按钮行；复制成功→图标转 ✓ + "已复制" 2s(state 驱动，去 querySelector)
                             纠错仅作用原文；译文 tab 时纠错隐藏/禁用
──────────────────────────
(A) 加载：  ▐▓▓▓▓░░░ 识别中…  [取消]
(B) 成功：  markdown 单层渲染（原文流式；译文完成后可切）
(C) 失败：  ┌──────────────────────────┐
           │ ⚠ 识别失败  (role=alert)  │   错误不写入 results
           │   网络错误，请重试         │   toast 同时弹一次
           │   [ ↻ 重试识别 ]          │   第2次起 prompt 末尾追加重试提示词
           └──────────────────────────┘
```

- 复制复制当前激活 tab 内容；`⎘/✎/↻` 均内联 SVG、`aria-label`、键盘可达、focus 环。
- 翻译失败：不污染原文，译文 tab 显「重试翻译」，与识别失败通道对称。
